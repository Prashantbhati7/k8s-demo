


# command to execute and make deployment using yaml file 
``` kubectl apply -f <yaml-file-name-absolute-or-relative-path/link-if-it-is-remote>```

#  to delete this deployment use this command 
 ``` kubectl delete -f <yaml-file-name-absolute-or-relative-path/link-if-it-is-remote> ```
 

# command to execute and make service using this file 
```  kubectl apply -f service-node-app.yml         ```

#  to delete this service use this command 
``` kubectl delete -f service-node-app.yml ```
